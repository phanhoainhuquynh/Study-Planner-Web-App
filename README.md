# Study Planner App
A simple frontend project built with **HTML, CSS, and JavaScript**.

## Features
- Add study tasks
- Set course, due date, and priority
- Mark tasks as completed
- Delete tasks
- Store tasks in localStorage
- Responsive layout

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Suggested CV entry
**Study Planner Web App (HTML, CSS, JavaScript)**  
- Developed a responsive web application to help students manage study tasks and homework  
- Implemented task creation, completion tracking, and deletion using JavaScript  
- Used localStorage to persist user data in the browser
